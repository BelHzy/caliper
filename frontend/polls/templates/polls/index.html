Caliper Benchmarking Report

Report Version & Date : v2.0-20160329

Caliper is a test framework focuses on benchmarking and performance evaluation of server platforms. It is developed primarily to evaluate various ARM64 server platforms. However, as a benchmarking platform, it caters and tested on other hardware platforms like x86. With Caliper you can test hardware and the software platform running on it easily, get visible results and compare the performance. It can help to discover issues related to hardware or software. To ease the first level of comparison of platforms, it provides simple score based view as the top most view. Scores are generated based on the original test results on the corresponding platform using uniform normalization algorithms across the platforms. If you want to know more about caliper, please visit Caliper page in Open Estuary Website

How to read the Scores in this report :
Higher the score, better the performance!
Ex: Usually lower the latency, it is better. However, in this report, higher the latency score, it is better!
All the scores are converted to percentiles. Means Max Score is 100.
In every score, one platform will score 100 and others will be relative percentile score to this.
Please read the report accordingly.

Summary Comments/Issues observed

This section will provide summary comments/high level issue observations based on the scores(focus w.r.t D02). This is only for reference to help further analysis or to improve the benchmarking report overall. These are not intended for any claim.
1. HP Moonshot(ARM64 and X86 platforms), AMD Overdrive, RH2288 platforms are added.
2. HP Moonshot system tested from HP Lab remotely. The network, harddisk provided(SSD) etc were of high quality. These factors would have influenced the good results from HP Moonshot in the overall scoring. [Anyways to be analysed each in detail]
3. There are some functional tests failed in most of the platforms. D02 has failed tests in kernel,dio, posix, mem, proc and cpu.
4. CPU functional test score D02 is the lowest
5. MOONSHOT-M700P(x86 based) has the HIGHEST overall score in algorithm,application,io,memory, disk and cpu
6. D02 has the LOWEST overall score in latency, network, application, memory and disk. Also low scores for algorithm, io and cpu. This is *alarming* and needs to analyse each in detail.
7. RH2288-E52690V3 has the HIGHEST overall score in latency and MOONSHOT-M400(ARM64) has the HIGHEST overall score for network
8. D02 is slightly ahead of AMD Overdrive(ARM64) and MOONSHOT-M400(ARM64) ONLY in algorithm
9. AMD Overdrive(ARM64) and MOONSHOT-M400(ARM64) scored very HIGH compared to D02 in latency, application and disk
10. MOONSHOT-M400(ARM64) has LOWEST overall score in algorithm and cpu. AMD Overdrive(ARM64) has LOWEST overall score in io.

Caliper Disclaimers:
 a) The actual test results/values are processed using different algorithm to derive the Scores. The Scores are intended only to compare various parameters among the platforms. If you want to see the original test results/values for debugging/analysis, please request for caliper test execution logs.
b) Each benchmark execution can depend on system conditions at the time of run. Hence, even if, caliper reboots the platform before each performance tests, the repeated test may not give 'exactly' same results for performance
c) There could be variations in the results due to specific peripheral hardwares in each platform(like RAM, HDD models etc).
d) These results are not intended for any claim, rather it is intended to use to improve the platforms as needed
e) During the initial analysis, we observed certain minor issues which demands score calculations optimization. We have marked as 'FixMe' with details, wherever needed.
f) Please provide your comments and feedback to caliper@open-estuary.org. It will help us to improve the report
